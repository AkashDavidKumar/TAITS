## 🛠️ Technology Stack & Site Features

This repository hosts the official website for TAITS Tech, recently refactored to achieve modern performance, responsive UX design, and complete cross-device support.

* **Frontend Stack**: Semantic HTML5, Vanilla JavaScript, and Responsive CSS (Variables, Flexbox, CSS Grid).
* **Interactive Elements**:
  - **Fluid Custom Cursor**: Built with GPU-accelerated translation and linear interpolation (`lerp`) for smooth tracking. Disabled on mobile/touch viewports for native accessibility.
  - **Interactive Particle Background**: Real-time canvas particle simulation that dynamically adjusts particle counts based on viewport size.
  - **Performance Optimization**: Fully integrated with the **Page Visibility API** to pause all animation loop execution when the page tab is hidden or minimized, conserving CPU and battery.
  - **Fully Responsive Navigation & Cards**: Redesigned static interactive wheel elements that transform dynamically into clear list structures on mobile devices.
